Goals

The main goal for this website is to improve the business of Petite Treats Bakery by increasing the number of new and repeat customers. It will show the available products of the bakery as well as show the items which are on specials. A newsletter signup form will also allow customers to sign up for the newsletter to remain informed about the bakery and its products.

Success Evaluation

3 months after the implementation of the website, the increase in the number of customers for those 3 months can be compared to the increase in the number of customers in the previous 3 months. If the increase in the number of customers in the previous 3 months is less than that in the 3 months following the implementation of the website, then the website implementation can be considered successful. A longer time period can be used for more accurate results.

Target Audience

Since customers of all ages already exist for the bakery, the website would be accessible to all ages. Site Flowchart

Content Analysis

  • Navigation
    • A horizontal navigation bar would be used at the top of the page which would be able to bring users to all the major pages.
    • Another row of navigation links would also be used at the bottom of the page for users who have scrolled down.
    • Navigation links:
      • Home
      • Menu
      • Specials
      • About
  • Home Page
    • Text to invite users to sign up for weekly specials, with a link that will send users to the specials page.
    • Some images of the products between the text.
    • Text which displays the comments of customers.
    • A form which collects some user information for the users that want to sign up for the weekly newsletter.
  • Menu Page
    • Some images beside a list of the available bakery items which will link to each of their individual pages.
    • The list will contain:
      • Donuts ($4.50)
      • Vanilla Slice ($4.50)
      • Randy Tart ($3.95)
      • Custard Tart ($3.95)
      • Raspberry Cheesecake ($5.50)
      • Apple Cake Slice ($5.50)
  • Specials Page
    • A list of 3-4 items which are on special, and each item in the list will link to their respective product page.
  • About Page
    • A brief description of the bakery, along with an image underneath it.
  • Footer
    • Contains navigation links as well as contact information:
      • Contacts: 0455 5555 555
      • Catering: 0433 3333 333
      • Address: 123 Prize Lane, Kirwan, QLD
